# astrbot_plugin_proactive_chat
# 主动聊天插件 for AstrBot

让机器人像真正的群友一样主动发起聊天。支持群聊/私聊独立配置、自定义人格、自定义话题主题、上下文延续，并内置详细日志和降级机制。

## ✨ 主要功能

- **群聊与私聊完全分离**：各自的开关、检查间隔、触发概率、白名单、话题权重独立设置。
- **自定义人格**：在插件配置中直接填写人格描述（优先于系统人设），留空则自动降级使用系统人设，再留空则使用默认人格。
- **多种话题来源**：
  - **历史消息**：根据最近聊天记录延续话题（需平台支持获取历史）。
  - **知识/时间**：根据当前季节、时间段生成话题。
  - **预置话题**：从 JSON 文件中随机选取完整消息（群聊/私聊分开）。
  - **自定义主题**：用户自定义主题列表（如“聊一聊最近的热点新闻”），LLM 根据主题生成具体内容。
- **LLM 生成与降级**：
  - 支持独立配置 LLM（OpenAI、DeepSeek、Anthropic、自定义），也支持回退到 AstrBot 系统默认 LLM。
  - 如果 LLM 生成失败或未配置，自动降级到预设话题，确保总能发言。
- **详细日志**：记录每次触发检查、话题选择、LLM 调用成功/失败及原因、降级情况，便于调试。

## 📦 安装

1. 将本插件文件夹放入 AstrBot 的 `addons` 目录。
2. 重启 AstrBot，在 WebUI 中启用插件。
3. 根据需要修改配置（重点：自定义人格、自定义主题列表、LLM 密钥等）。
4. 确保已安装依赖：
   ```bash
   pip install apscheduler openai
   # 如果使用 Anthropic 还需：
   pip install anthropic
